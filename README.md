# EiRA
The source code and data of EiRA

# Pre-requisite:
- Python3, numpy, pandas, pytorch(2.6.0+cu118),peft(0.14.0)
- esm3([https://github.com/facebookresearch/esm](https://github.com/evolutionaryscale/esm))

# Installation:
- 1.Download the source code in this repository.
- 2.Download the weights of EiRA at https://huggingface.co/zengwenwu/ESM-DBP, and make sure they locate in the same folder.
- 3.Typical install time on a "normal" desktop computer is about about 30 minutes, depending on the Download speed from Huggingface.
